气泵
; （不稳定，易崩） 模块：气泵，状态类型 ―― AE生效期间，单位咱蹦高了，画一条抛物线
Pump.Range=0 ;冲击力，飞多远，单位格
Pump.Gravity=0 ;自定义重力，默认使用全局重力
Pump.PowerBySelf=no ;由弹头附加的状态，会记录下弹头的爆炸位置，飞多远不计算爆炸位置和单位之间的距离衰减
Pump.Lobber=no ;高抛
Pump.Inaccurate=no ;落点随机
Pump.ScatterMin=0.0 ;最小散布范围，单位格
Pump.ScatterMax=0.0 ;最大散布范围，单位格，默认为全局BallisticScatter
; 特殊设置，步兵起飞后显示的帧序列
Pump.InfSequence=Ready
; 对象影响过滤器专属设置
Pump.AffectTypes=none ;类型过滤，可影响的单位，如果设置，则向列表中的单位复仇
Pump.NotAffectTypes=none ;类型过滤，不可影响的单位，如果设置，则不向列表中的单位复仇，优先级更高
; 对象影响单位设置
Pump.AffectInfantry=yes ;类型过滤，影响步兵
Pump.AffectUnit=yes ;类型过滤，影响载具
Pump.AffectAircraft=yes ;类型过滤，影响飞机
; 影响特定对象
Pump.AffectInAir=yes ;影响在空中的单位
Pump.NotAffectMarks=none ;不影响带有这些标记的对象
Pump.OnlyAffectMarks=none ;只影响带有这些标记的对象
; 影响对象敌我识别专属设置
Pump.AffectsOwner=yes ;是否影响同阵营的
Pump.AffectsAllies=yes ;是否影响友军
Pump.AffectsEnemies=yes ;是否影响敌人
Pump.AffectsCivilian=yes ;是否影响中立目标
