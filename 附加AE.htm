附加AE

[TechnoType]
[ProjectileType]
[WarheadType]
AttachEffectTypes=AutoWeapon1,StandTest1 ;要挂载的AE类型，写弹头上表示被该弹头伤害到的单位挂载
AttachEffectChances=100%,100% ;附加效果的成功率
AttachToSource=no ;不附加给目标，而是反过来附加给攻击者
AttachFullAirspace=no ;使用弹头附加时，搜索范围改为圆柱体
StandTrainCabinLength=512 ;火车类型的替身的车厢长度，影响替身之间的间隔
AffectInAir=yes ;使用弹头附加效果是否影响空中单位的总开关
AffectShooter=yes ;使用弹头附加效果时是否影响武器的发射者

; 通过乘客给载具附加AE，载具端写AE组，可以设置多组，X=[0-127]，乘客带指定的AEMode进入时激活，类似FV，进入多个同Mode不会叠加，踢出乘客后，现存的乘客中不存在该Mode则移除对应的AE
[TechnoType]
AttachEffectTypesX=AutoWeapon1,StandTest1 ;要挂载的AE类型
AttachEffectTypesX.AttachByPassenger=yes ;仅由乘客身上的AEMode激活赋予
AEMode=-1 ;写在乘客上，指定激活载具上的哪一组AE赋予

; 通过乘客给载具附加AE，乘客端写AE组，可以设置多组，X=[0-127]，不写等同于0，进入载具为载具附加AE
[TechnoType]
UploadX.AttachEffects=AutoWeapon1,StandTest1 ;为载具附加的AE
UploadX.SourceIsPassenger=yes ;该组AE来源为乘客，不同来源会叠加，乘客被踢出时移除AE
; 对象影响过滤器专属设置
UploadX.AffectTypes=none ;类型过滤，可影响的单位，如果设置，则只可以影响列表中的单位
UploadX.NotAffectTypes=none ;类型过滤，不可影响的单位，如果设置，则不影响列表中的单位，优先级更高
; 对象影响单位设置
UploadX.AffectUnit=yes ;类型过滤，影响载具
UploadX.AffectAircraft=yes ;类型过滤，影响飞机
; 影响特定对象
UploadX.AffectStand=no ;影响替身
UploadX.AffectInAir=yes ;影响在空中的单位
UploadX.NotAffectMarks=none ;不影响带有这些标记的对象
UploadX.OnlyAffectMarks=none ;只影响带有这些标记的对象

; 通过发射武器为自身附加AE，可以设置多组，X=[0-127]，不写等同于0，在载具内发射时不会附加
; 对AttachFire同样有效，作用于AttachFire时的目标是射手而不是武器持有者
; 当AttachFire的武器是从抛射体上射出时，附加的对象是抛射体而非抛射体的持有人
; 当AttachFire的武器是从载具内射出时，附加的对象是载具而非乘客
[WeaponType]
FeedbackX.AttachEffects=AutoWeapon1,StandTest1
FeedbackX.AttachChances=100%,100% ;附加效果的成功率
FeedbackX.AttachToTransporter=no ;在载具内发射时，可为载具附加，否则不会附加，如果是AttachFire，则作用对象必定为载具
; 对象影响过滤器专属设置
FeedbackX.AffectTypes=none ;类型过滤，可影响的单位，如果设置，则只可以影响列表中的单位
FeedbackX.NotAffectTypes=none ;类型过滤，不可影响的单位，如果设置，则不影响列表中的单位，优先级更高
; 对象影响单位设置
FeedbackX.AffectTechno=yes ;类型过滤，影响单位（注意Affect没有s）
FeedbackX.AffectBuilding=yes ;类型过滤，影响建筑
FeedbackX.AffectInfantry=yes ;类型过滤，影响步兵
FeedbackX.AffectUnit=yes ;类型过滤，影响载具
FeedbackX.AffectAircraft=yes ;类型过滤，影响飞机
; 对象影响抛射体设置
FeedbackX.AffectBullet=yes ;类型过滤，影响抛射体（注意Affect没有s）
FeedbackX.AffectMissile=yes ;类型过滤，影响ROT>0的抛射体
FeedbackX.AffectTorpedo=yes ;类型过滤，影响Level=yes的抛射体
FeedbackX.AffectCannon=yes ;类型过滤，影响Arcing=yes的抛射体
FeedbackX.AffectBomb=yes ;类型过滤，影响Vertical=yes的抛射体
; 影响特定对象
FeedbackX.AffectStand=no ;影响替身
FeedbackX.AffectInAir=yes ;影响在空中的单位
FeedbackX.NotAffectMarks=none ;不影响带有这些标记的对象
FeedbackX.OnlyAffectMarks=none ;只影响带有这些标记的对象
; 对象敌我识别专属设置
FeedbackX.DeactiveWhenCivilian=no ;启用时，当所属是平民时，或者被狙掉了驾驶员，不会激活效果
