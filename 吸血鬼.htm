Help > AttachEffect系统 > 【单位专用】模块 >
吸血鬼
; 模块：吸血鬼 ―― AE生效期间，附着对象对其他单位造成的伤害，按比例返回
Vampire.Chance=100% ;触发吸血的概率
Vampire.Percent=0% ;吸血的比例
Vampire.TriggeredTimes=-1 ;可触发的次数，次数用完立刻强制结束效果
; 吸血目标对象影响过滤器专属设置
Vampire.AffectTypes=none ;类型过滤，可影响的单位，如果设置，则向列表中的单位复仇
Vampire.NotAffectTypes=none ;类型过滤，不可影响的单位，如果设置，则不向列表中的单位复仇，优先级更高
; 吸血目标对象影响单位设置
Vampire.AffectBuilding=yes ;类型过滤，影响建筑
Vampire.AffectInfantry=yes ;类型过滤，影响步兵
Vampire.AffectUnit=yes ;类型过滤，影响载具
Vampire.AffectAircraft=yes ;类型过滤，影响飞机
; 影响特定目标对象
Vampire.AffectInAir=yes ;影响在空中的单位
Vampire.NotAffectMarks=none ;不影响带有这些标记的对象
Vampire.OnlyAffectMarks=none ;只影响带有这些标记的对象
; 吸血目标对象敌我识别专属设置
Vampire.AffectsOwner=yes ;是否影响同阵营的
Vampire.AffectsAllies=yes ;是否影响友军
Vampire.AffectsEnemies=yes ;是否影响敌人
Vampire.AffectsCivilian=yes ;是否影响中立目标
Vampire.DeactiveWhenCivilian=no ;启用时，当所属是平民时，或者被狙掉了驾驶员，不会激活效果
